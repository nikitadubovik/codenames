# Codenames — мультиплеер для игры с друзьями

Реалтайм-версия «Кодовых имён»: комнаты по коду, отдельный вид для капитанов
и игроков, пометки кликом, подтверждение двойным кликом, вся логика ходов.

## Локальный запуск

```bash
npm install
npm start
# открыть http://localhost:3000 в нескольких вкладках/устройствах
```

Все, кто ввёл **одинаковый код комнаты**, попадают в одну игру.

## Как играть

1. Каждый вводит имя и общий код комнаты.
2. Выбирает сторону и роль: **Капитан** (видит цвета карт) или **Игрок**.
3. Капитан ходящей команды даёт подсказку: **слово + число (0–9)**.
4. Игроки этой команды:
   - **один клик** по карте — пометка (её видят все),
   - **двойной клик** — открыть карту.
5. Своя карта → можно продолжать (до `число + 1` попыток).
   Чужая/нейтральная → ход переходит. Чёрная → мгновенное поражение.
6. «Завершить ход» — остановиться досрочно.
7. Кто первым открыл все свои карты — победил.

## Деплой (бесплатно, из этого же репозитория)

GitHub Pages не подойдёт — нужен работающий сервер (WebSocket).
Подключите репозиторий к любому из:

- **Render.com**: New → Web Service → ваш репозиторий →
  Build `npm install`, Start `npm start`. Порт берётся из `process.env.PORT`.
- **Railway.app** / **Fly.io** / **Glitch** — так же, одна команда старта.

Ссылку с хостинга отправляете друзьям — играете из браузера.

## Что можно докрутить

- Проверка, что у каждой команды ровно один капитан.
- Таймеры на ход, звук, история партий.
- Больше слов в `words.js` (сейчас 130).
- Правило клуба «0 = бесконечные попытки» (сейчас `k+1`, по вашей спецификации).
